# TicTacToe Game

Desktop TicTacToe game developed using Java Swing in NetBeans IDE.

## Features
- Interactive graphical user interface
- Two-player turn-based gameplay
- Win and draw detection system
- Score tracking for both players
- Game timer functionality
- New Game, Reset, and Exit options
- Dynamic board updates during gameplay

## Technologies Used
- Java
- Java Swing
- NetBeans IDE
